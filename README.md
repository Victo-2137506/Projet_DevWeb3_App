## Procédure d'installation de l'application sur un poste local

- Copier le fichier .env.example et créer un fichier .env à la racine du projet.

- Modifier la variable VITE_API_URL afin qu’elle pointe vers l’URL de l’API (locale ou en ligne).

- Démarrer le projet en exécutant la commande "npm run dev".

## Description sommaire de l'application publiée

Le site est une base de données en ligne de personnages historiques.
L’accès au site nécessite une authentification préalable.
La page d’accueil présente une introduction au site.
Le menu permet de naviguer entre les différentes pages, de changer la langue et de se déconnecter.
La page « Histoire » affiche la liste de tous les personnages historiques disponibles dans la base de données.
Une barre de recherche permet de filtrer les personnages par pays et/ou par siècle.
Le bouton « Ajouter un personnage historique » ouvre un formulaire pour ajouter un nouveau personnage.
En cliquant sur un personnage historique, ses informations détaillées s’affichent.
Cette page comporte également des boutons permettant de le supprimer, de le modifier ou de revenir à la liste.
